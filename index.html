FastTrack Ruby on Rails

This two-day intensive training gives you hands-on experience with creating data-driven website using the popular Ruby on Rails framework, and will be ready to tackle your own web project.

Starting with the Ruby and Rails basic, you will learn the full Rails MVC stack. You will work through the full life-cycle of a Rails project, giving you a first-hand experience creating a Rails app.

Duration

2 days

Course Outline

By the end of the training, you will have a firm understanding of Ruby on Rails concepts like:

  • MVC (Model, View, Controller) architecture
  • Bundler
  • Rails console
  • Rake
  • Creating Rake tasks
  • Rails' configuration files
  • Generators
  • RESTful routes
  • Migrations
  • ActiveRecord models
  • Model validations
  • Model associations
  • Model callbacks
  • AREL
  • ActionController controllers
  • Filters
  • Metal/Rack
  • ERB based views
  • HAML based views
  • Rails plugins
  • Unit testing

Pre-requisites

All participants of the training course must be familiar with the general principles of object oriented programming (OOP), with some exposure to the following concepts and technologies:

  • Web development principles
  • Basic knowledge of HTML, CSS and JavaScript
  • Basic understanding of HTTP protocols (GET, POST, redirection, SSL)
  • Basic knowledge of SQL
  • Familiarity with relational databases (MySQL, SQLite, etc.)
